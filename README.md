## 📍 Overview

DevX Studio is designed to help businesses grow by offering a suite of digital services including Product Design, Marketing, and Web Development. The website features a "Dark Mode" aesthetic (implied by typical modern agency styles) with high-contrast elements, smooth scrolling, and testimonial carousels.

**Live Demo:** https://devxstudio-agency.netlify.app/

---

## ✨ Key Features

* **Responsive Navigation:** A fully functional navbar with a hamburger menu for mobile devices.
* **Hero Section:** innovative headline with Call-to-Action (CTA) buttons and video integration hooks.
* **Service Showcase:** A grid-based layout displaying core services using **Remix Icons**.
* **Interactive Testimonials:** A smooth, touch-friendly customer feedback slider powered by **Swiper.js**.
* **Scroll Animations:** Elements fade in and slide up upon scrolling using **ScrollReveal**.
* **Portfolio Metrics:** A dedicated section highlighting years of experience and project stats.
* **Newsletter Subscription:** A styled input form for user engagement.
* **Cross-Browser Compatible:** Tested on Chrome, Firefox, and Edge.

## 🛠️ Tech Stack

This project is built using pure frontend technologies with external libraries for enhanced interactivity.

| Technology | Usage |
| :--- | :--- |
| **HTML5** | Semantic structure and markup. |
| **CSS3** | Flexbox, Grid, Media Queries, and custom variables. |
| **JavaScript (ES6)** | DOM manipulation and library initialization. |
| **Remix Icon** | Professional icon set (via CDN). |
| **Swiper.js** | Touch-enabled mobile slider for testimonials. |
| **ScrollReveal** | JavaScript library for scroll animations. |

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites
You need a modern web browser and a code editor (like VS Code).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/hamayldev/DevX-Studio.git](https://github.com/hamayldev/DevX-Studio.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd DevX-Studio
    ```

3.  **Run the project:**
    * Simply open the `index.html` file in your browser.
    * **Recommended:** Use the "Live Server" extension in VS Code for real-time updates.
